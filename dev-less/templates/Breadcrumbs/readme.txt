Строка навигации может содержать ряд различных свойств. Google распознает следующие свойства строки навигации.
title 	Название элемента навигации.
url 	URL элемента навигации.
child 	Следующий элемент навигации в иерархии. Дочерний элемент также должен быть элементом навигации.

Если на странице имеется только одна строка навигации, то Google будет придерживаться того порядка элементов, который показан на странице. Если же строк навигации несколько, то придется указать, к какой из них относится каждый элемент навигации. Для этого нужно определить иерархию элементов с помощью свойства child
